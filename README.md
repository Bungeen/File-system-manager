# File System Manager

---

Консольный файловый менеджер с расширяемой архитектурой на **.Net (C#)**. Позволяет выполнять базовые операции над любой поддерживаемой файловой системой (по умолчанию присутствует реализация для **локальной файловой системы**)

---

## ⭐ Особенности

* Соблюдение принципов **SOLID** и **GRASP**: Приложение строго разделено на слои. Бизнес логика и логика парсера независимы.
* Dependency Injection: Приложение использует интерфейсы, что позволяет добавлять свои реализации файловых систем, команд и прочего.
* OCP: Парсер команд основан на Цепочках ответственности, что позволяет без трудностей добавлять/удалять/изменять команды и их аргументы.
* Использование классических паттернов проектирования
* Не зависит от сторонних библиотек
* Ленивое выполнение операций
* Поддержка навигации по файловой системе с использованием `.` `..`

---



## 📚 API Reference

Управление подключением
* `connect [Address] [-m Mode]` — Подключение к файловой системе по абсолютному пути. Задает корень рабочей области.
* `disconnect` — Отключение от текущей ФС. Блокирует остальные команды до нового подключения.
 
Навигация и просмотр

* `goto [Path]` — Переход по указанному пути (поддерживаются относительные и абсолютные пути).
* `tree list {-d Depth}` — Вывод дерева каталогов относительно текущей позиции. `-d`: Глубина рекурсии (по умолчанию 1). 

Параметры отображения (символы ветвления) вынесены в конфигурацию.

Операции с файлами
* `file show [Path] {-m Mode}` — Просмотр содержимого файла.
* `file copy [Source] [Destination]` — Копирование файла.
* `file move [Source] [Destination]` — Перемещение файла.
* `file delete [Path]` — Удаление файла.
* `file rename [Path] [NewName]` — Переименование файла (без изменения пути).
